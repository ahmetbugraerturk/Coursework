This file contains materials for one instance of the attacklab.

Files:

    ctarget

Linux binary with code-injection vulnerability.  To be used for phases
1-3 of the assignment.

     cookie.txt

Text file containing 4-byte signature required for phases 1-3 of the assignment

     farm.c

Source code for gadget farm present for phases 1-3 of the assignment.  You
can compile (use flag -Og) and disassemble it to look for gadgets.

     hex2raw

Utility program to generate byte sequences.  See documentation in lab
handout.

    bonus/rtarget

Linux binary with return-oriented programming vulnerability.  To be
used for the bonus part of the assignment.

    bonus/farm.c

Source code for gadget farm present in this instance of bonus/rtarget.  You
can compile (use flag -Og) and disassemble it to look for gadgets.

     bonus/cookie.txt

Text file containing 4-byte signature required for the bonus part.

     bonus/hex2raw

Same executable as hex2raw in the root folder, copied here for convenience.
